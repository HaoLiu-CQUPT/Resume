# 个人简历网页

一个现代化、响应式的个人简历网页项目，采用纯 HTML、CSS 和 JavaScript 开发，无需任何框架依赖。

## 📋 项目简介

这是一个单页面的个人简历展示网站，具有美观的界面设计和流畅的交互体验。网站采用左右分栏布局，左侧为固定导航栏，右侧为主要内容区域，包含了个人信息、教育经历、项目经历、工作经历、个人技能和作品集等模块。

## ✨ 功能特性

- 🎨 **现代化设计**：采用渐变色配色方案，界面美观大方
- 📱 **响应式布局**：完美适配桌面端、平板和移动设备
- 🔄 **交互功能**：
  - 平滑滚动导航
  - 项目详情展开/折叠
  - 导航栏自动高亮当前章节
- ⚡ **性能优化**：纯原生代码，加载速度快
- 🎯 **易于定制**：代码结构清晰，便于修改和扩展

## 🛠️ 技术栈

- **HTML5**：语义化标签，结构清晰
- **CSS3**：
  - CSS Grid 布局
  - Flexbox 布局
  - CSS 动画和过渡效果
  - 响应式媒体查询
- **JavaScript (ES6+)**：
  - DOM 操作
  - 事件处理
  - 平滑滚动
  - 性能监控

## 🚀 使用方法

### 本地运行

1. **克隆或下载项目**
   ```bash
   # 如果使用 Git
   git clone <repository-url>
   cd Resume
   ```

2. **直接打开**
   - 双击 `index.html` 文件在浏览器中打开
   - 或使用本地服务器运行（推荐）：
     ```bash
     # 使用 Python
     python -m http.server 8000
     
     # 使用 Node.js (需要安装 http-server)
     npx http-server
     
     # 使用 PHP
     php -S localhost:8000
     ```
   - 然后在浏览器中访问 `http://localhost:8000`

### 部署

可以将项目部署到以下平台：

- **GitHub Pages**：将代码推送到 GitHub 仓库，启用 Pages 功能
- **Netlify**：拖拽项目文件夹到 Netlify 即可部署
- **Vercel**：连接 GitHub 仓库自动部署
- **其他静态网站托管服务**

## 📝 自定义说明

### 修改个人信息

编辑 `index.html` 文件中的以下部分：

1. **个人信息**（第 28-44 行）：
   - 修改姓名、简介、联系方式等
   - 替换头像图片链接

2. **教育经历**（第 47-61 行）：
   - 更新学校名称、专业、时间
   - 修改描述内容

3. **项目经历**（第 86-124 行）：
   - 添加或修改项目信息
   - 更新技术栈和成果描述

4. **工作经历**（第 127-150 行）：
   - 修改公司名称、职位、时间
   - 更新工作描述和职责

5. **个人技能**（第 153-189 行）：
   - 添加或删除技能标签
   - 调整技能分类

6. **作品集**（第 192-220 行）：
   - 替换作品图片
   - 更新作品描述和链接

### 修改样式

编辑 `styles.css` 文件：

- **主色调**：修改 `#667eea` 和 `#764ba2` 颜色值（渐变紫色主题）
- **字体**：修改 `font-family` 属性
- **布局**：调整 `.container` 的 `grid-template-columns` 来改变侧边栏宽度
- **间距**：修改各个元素的 `padding` 和 `margin` 值

### 添加新功能

在 `script.js` 文件中可以添加新的交互功能，例如：
- 添加更多动画效果
- 集成第三方服务
- 添加表单提交功能
